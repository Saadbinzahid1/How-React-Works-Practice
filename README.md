# ⚛️ How React Works – Behind the Scenes

This repository is dedicated to exploring and understanding how React works internally.  
Instead of only building applications, this project focuses on learning the core mechanisms that power React.

The goal is to experiment with different concepts and observe how React handles rendering, state updates, and component behavior behind the scenes.

---

## 🎯 Objectives

- Understand how React renders components
- Explore how state updates trigger re-renders
- Learn about the Virtual DOM
- Observe React's reconciliation process
- Study component lifecycle behavior
- Experiment with React Hooks

---

## 🧪 Experiments Included

This repository contains small experiments and examples such as:

- Component rendering behavior
- State updates and batching
- Re-render triggers
- useState and useEffect behavior
- Key props and list rendering
- Performance observations

Each example focuses on a specific concept to clearly demonstrate how React works internally.

---

## 🛠️ Technologies Used

- React
- JavaScript (ES6+)
- HTML5
- CSS3

---

## 🧠 Learning Purpose

This project is purely for **learning and experimentation**.  
It helps build a deeper understanding of React’s architecture and internal workflow, which is essential for writing efficient and optimized React applications.

---

## 📌 Note

The examples in this repository are intentionally simple and focused.  
The purpose is to clearly observe React’s internal behavior rather than build full applications.
